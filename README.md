# 针对即梦、可灵、Happy Horse、Vidu 的能力测评

针对四款音视频生成模型进行系统评测，持续记录它们在生成质量、指令遵循、音画同步、稳定性和不同应用场景下的能力表现。

## 在线报告

- [查看完整评测报告](https://givenwels.github.io/Jimeng-Kling-Happy-Horse-Vidu-Evaluation/)
- [查看 HTML 源文件](./index.html)

## 评测对象

- 即梦 Jimeng
- 可灵 Kling
- Happy Horse
- Vidu

## 研究目的

- 比较不同模型的综合能力与稳定性
- 分析各模型在不同场景和评测维度中的优势与短板
- 记录典型 Bad Case 与能力边界
- 为后续模型迭代和选型提供可追踪的研究依据

## 当前结论

即梦综合领先，可灵稳定第二，Happy Horse 音频能力和上限突出但波动较大，Vidu 更适合低成本入门。
